组织的开源成熟度

开源与政策界的交汇点

 • 欧洲开放论坛（OFE）是一个总部位于布鲁塞尔的非营利性独立智库，致力于向欧洲各地的政策制定者和社区阐释计算领域开放性的优势。它支持一个由开放论坛学院（OpenForum Academy）研究员组成的独立全球网络，并主办以研究为重点的开放论坛学院研讨会，汇聚全球专注于开源、政策和技术影响的研究人员。欧洲开放论坛与欧盟委员会、欧洲议会以及国家和地方政府密切合作。每年，欧洲开放论坛主办方都会举办欧盟开源政策峰会，这一独特活动填补了欧洲最高层政策制定与开源社区和企业代表之间的鸿沟。更多信息请访问：https://summit.openforumeurope.org/



开源软件资助（资金）

• GitHub 赞助者 （允许）让开发者社区能够直接在 GitHub 上为他们所依赖的开源软件项目提供财务支持，详情请看网址：https://github.com/sponsors 

• GitHub 安全开源基金，详情请看网址：https://resources.github.com/github-secure-open-source-fund/

 • thanks.dev 是一个平台，可自动将公司和开发者每月的捐赠分配到其开源依赖树中，从而轻松为所有他们所依赖的项目的维护者提供财务支持，而不仅仅是那些最受欢迎的项目。更多信息请访问：https://thanks.dev/static/why 

• Ben Hoyt，“Canonical + thanks.dev = 回馈开源开发者”，2025年5月，网址：https://canonical.com/blog/canonical-thanks-dev-giving-back-to-open-source-developers 

• 德国主权技术机构投资于我国经济和社会开放数字基础设施的开发、改进和维护。该机构包括多个项目，其中主权技术基金资助关键开源软件项目，主权技术韧性项目支持关键开源软件项目的安全维护，坚守的维护者奖学金项目资助关键开源软件项目的维护者。更多信息请访问：https://www.sovereign.tech/ 

• 欧盟委员会的下一代互联网（NGI）倡议是地平线欧洲（Horizon Europe）下的公共资助倡议，支持研究和开发支持信任互联网的开放互联网技术。NGI倡议在2019年至2024年期间为1200多个项目提供了1.4亿欧元的资金，并为2024年至2027年额外拨款3200万欧元。更多信息请访问：https://ngi.eu/ 

• 数字基础设施洞察基金是由福特基金会、阿尔弗雷德·P·斯隆基金会、奥米迪亚网络、施密特未来基金和开放集体共同资助的多方资助者倡议，旨在资助研究，以更好地理解开放数字基础设施的构建和部署方式。 更多信息请访问：https://infrastructureinsights.fund/ 

 • Sam Boysel、Frank Nagle、Hilary Carter、Anna Hermansen、Kevin Crosby、Jeff Luszcz、Stephanie Lincoln、Daniel Yue、Manuel Hoffmann、Alexander Staub。“2024年开源软件融资报告”。2024年11月。

关于作者

MARCO GEROSA是北亚利桑那大学计算机科学系的全职教授，同时也是LF Research的研究分析师。他在软件工程和开源软件方面的研究已在顶级期刊上发表了300多篇论文。他担任知名会议的程序委员会成员，并为多家期刊担任审稿人。Gerosa博士拥有博士学位、信息学硕士学位和计算机工程学士学位。他是电气和电子工程师学会（IEEE）和国际计算机学会（ACM）的高级会员。他指导过多名博士和硕士生，这些学生如今已是顶尖机构的研究人员。此外，他还有20多年的教学经验。欲了解更多信息，请访问http://www.marcoagerosa.com。
 Adrienn Lawson 现任Linux基金会定量研究总监，负责领导数据驱动的项目，以深入了解开源生态系统。她拥有牛津大学的社会数据科学专业知识，并拥有学术和政府研究的背景，为分析分布式协作网络带来了方法论上的严谨性。在Linux基金会， Adrienn领导一个团队，对不同行业垂直领域和地理区域进行横断面研究，以提供对开源动态的全面洞察。她的工作包括对监管合规性、人工智能的影响以及可持续资助模式的实证调查。她提出基于证据的建议，为开源社区内的战略决策提供信息。
